Quick Start

terminal
# Install gRPC tools
pip install grpcio grpcio-tools numpy

# Clone and generate stubs
git clone https://github.com/Pragadeesh-19/HammingStore
cd HammingStore
python -m grpc_tools.protoc \
  -I src/main/protobuf \
  --python_out=. \
  --grpc_python_out=. \
  src/main/protobuf/hammingstore.proto
client.py
import grpc, numpy as np
import hammingstore_pb2 as pb
import hammingstore_pb2_grpc as pb_grpc

# Connect to the live instance
channel = grpc.secure_channel(
    "hammingstore.fly.dev:443",
    grpc.ssl_channel_credentials()
)
client = pb_grpc.HammingStoreStub(channel)

# Store a float embedding (server binarises it)
embedding = np.random.randn(384).astype(np.float32)
client.StoreFloat(pb.StoreFloatRequest(
    entity_id=1,
    float_vec=embedding.tobytes()
))

# Multi-hop chain traversal
results = client.QueryChain(pb.QueryChainRequest(
    start_entity_id=1,
    relation_ids=[101, 202],
    k=10
))

gRPC API

StoreFloat
Binarise and store a float embedding via random projection
StoreBinary
Store a pre-binarised 10,048-bit hypervector directly
StoreTypedEdge
Store a role-encoded typed relation between entities
SearchFloat
Approximate nearest-neighbour search via HNSW
QueryHop
Single typed-edge traversal: given (subject, relation) → objects
QueryChain
Multi-hop chain traversal across N relation types
QueryAnalogy
A:B :: C:? via XOR binding in hypervector space
Retract
Tombstone an entity, preventing further writes
